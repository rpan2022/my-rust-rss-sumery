1. **Rust Foundation Maintainers Fund Launch**: The Rust Foundation has officially launched the Maintainers Fund, established via RFC #3931. It creates a Funding team and a "Maintainer in Residence" program to financially support existing Rust project maintainers (e.g., compiler, standard library, Cargo). The goal is to provide stable, long-term funding for critical maintenance work. Individuals can donate via GitHub Sponsors, and companies can also contribute. This addresses the need for diversified funding as some maintainers lose corporate support.

2. **Rust 1.96.0 Released**: The new stable version introduces several features:
   - New `core::range` types (Range, RangeFrom, RangeInclusive) that are `Copy` and implement `IntoIterator` instead of `Iterator`, making them more ergonomic for indexing.
   - `assert_matches!` and `debug_assert_matches!` macros for pattern-matching assertions with better diagnostic output.
   - WebAssembly targets now default to not passing `--allow-undefined`, causing linker errors for undefined symbols to catch bugs earlier.
   - Stabilized APIs include `assert_matches!`, `debug_assert_matches!`, and various range-related types.
   - Two CVE fixes for Cargo regarding symlink extraction and URL normalization (not affecting crates.io users).
